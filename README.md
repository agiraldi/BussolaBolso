## 🚀 Projeto Bussola do Bolso

Sistema completo com:
- Backend Node.js
- Frontend Flutter Web
- Mobile Flutter

## 🧱 Arquitetura

Monorepo com Git Flow

## 🔗 Integrações
- Jira
- CI/CD (GitHub Actions)

## ▶️ Como rodar

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
flutter run -d chrome

### Mobile
cd mobile
flutter run
